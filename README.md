# 🔮 La Mágica Bruja Efsun — Sitio Web Oficial

Sitio web profesional para **La Mágica Bruja Efsun**, bruja, tarotista y consultora esotérica con atención presencial en A Coruña y online a todo el mundo.

---

## Características

- **Hero animado** con canvas de bosque encantado y cursor varita mágica
- **Sección de servicios** — Tarot, rituales de amor, limpieza energética, carta astral, pareja y abundancia
- **Tienda esotérica** — Velas rituales, cristales, kits y amuletos
- **Horóscopo personalizado con IA** — Lectura astral generada por inteligencia artificial
- **Sección de redes sociales** — Instagram y TikTok
- **Sección de contacto** — WhatsApp, Instagram, TikTok
- **Footer completo** con navegación y créditos
- Diseño **responsive** adaptado a móvil y escritorio
- Animaciones y efectos visuales con JavaScript vanilla

---

## Tecnologías

- HTML5
- CSS3 (variables, animaciones, grid, flexbox)
- JavaScript vanilla
- Google Fonts
- GitHub Pages (deploy)
